## Mission

> **To multiply human skill through sovereign technology—transforming spoken intent into trusted, buildable reality.**

Sovereign Dominion is not a landscaping app. It is the operating system for the built world that amplifies human capability, never replaces it. Every line of code is written to create more jobs, complete more projects, train more workers, and build an economy where technology serves the craftsperson.

## What Is Sovereign Dominion?

Sovereign Dominion is a mobile‑first, offline‑capable PWA (with native iOS/Android shells) that lets you point a smartphone camera at any physical space and instantly interact with photorealistic, editable 5D reconstructions. Through natural conversation with **Esther** — a warm, emotionally expressive voice agent — you can:

- 🎤 **Design by voice** — landscapes, hardscapes, structures, nine skilled trades
- 🖼️ **See photorealistic AR** — 5D Gaussian Splatting + NeRF at 90‑120 fps
- 🛒 **Get live local prices** — Home Depot, Lowe's, Menards, SiteOne
- 📜 **Check building codes** — violations glow red in AR
- 📄 **Export BIM blueprints** — IFC, DXF, DWG, PDF/A with cryptographic proof
- 🔍 **See underground** — X‑ray vision for utilities
- 🚁 **Monitor progress** — drone integration with change detection
- 👷 **Train workers** — AR/VR safety training in 10 languages
- 🔐 **Trust everything** — post‑quantum ZKP verifies every claim

## Quick Start

```bash
git clone https://github.com/jaetierce31/sovereign-dominion.git
cd sovereign-dominion
npm install
npm run dev
```

Open in Chrome Canary (enable WebGPU) or Safari on iOS 17+. Allow camera & microphone. Speak: "Esther, build a curved retaining wall 20 feet long in weathered basalt."

## Technology Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React + TypeScript, Tailwind CSS, Vite (PWA) |
| Rendering | 5D Gaussian Splatting + NeRF, WebGPU, WebGL2 fallback |
| Voice | Chatterbox TTS, Qwen3‑TTS (10 languages), neural ODE prosody, Whisper ASR |
| AI | ONNX Runtime (SegFormer, MobileNetV2, MiDaS, Wav2Vec2‑emotion) |
| Trust | QSSM lattice‑based ZKP (WASM), Ethereum/Polygon smart contracts, WebAuthn |
| Data | Automerge CRDT, IPFS (Helia), SQL.js, MapLibre GL JS |
| Mobile | Capacitor, ARKit, ARCore, custom LiDAR plugin |
| Deployment | Cloudflare Pages & Workers, Hardhat |
